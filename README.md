# Text2SQL

🔥 **Live Demo:** [https://text2sql-ag58.onrender.com/](https://text2sql-ag58.onrender.com/)

A highly aesthetic, fully local web application powered by **Google's Gemini** and the **Agno Framework** that autonomously converts natural language questions into precise, executable PostgreSQL database queries.

## Overview
This tool has been extensively upgraded to embrace true Agentic Architecture. Instead of relying on rigid chunks of database schemas fetched via a vector store, Text2SQL Studio runs an active **Agno Agent** equipped with granular schema lookup tools. 

When triggered, the Gemini-powered agent autonomously utilizes two scalable Python-based tools (`list_all_tables` and `get_table_ddl`) to dynamically interrogate your PostgreSQL database. This prevents hitting context-window limits on massive datasets, allowing the agent to target solely the exact tables it needs to deduce foreign keys and output perfect SQL logic directly to your interface!

## Project Structure
- `api.py` - The main execution entry point (FastAPI Server) providing a flawless synchronous pipeline without event-loop overhead against the `/api/generate-sql` route.
- `createEmbeddings.py` - The core engine that runs the Agno LLM Agent and the custom schema interrogation tools. *(Legacy filename maintained for project history).*
- `exportSchema.py` - A configurable script that connects directly to your local PostgreSQL instance and exports the table constraints statically into `schema.json`.
- `schema.json` - Your autogenerated database blueprint mapping.
- `static/` - Contains the premium modern frontend (HTML, raw JS, Vanilla CSS block) providing the responsive UI.
- `examples/` - Sandbox examples evaluating SQL outputs and raw integrations.
- `tests/` - Miscellaneous diagnostic scripts to isolate native engine flows independent of the Web UI.

## Environment Variables
The application requires a valid `.env` configuration file to instantiate perfectly. Ensure you define:
```env
GEMINI_API_KEY="your-google-ai-studio-api-key"
```

## Setup & Run locally

### 1. Requirements
Install all dependencies cleanly via pip:
```bash
pip install -r requirements.txt
```
*(Make sure to verify your Postgres connection details defined at the top of `exportSchema.py` match your setup!)*

### 2. Run the App
Launch the FastAPI application locally via Python:
```bash
python -m uvicorn api:app --reload
```

### 3. Usage
Navigate your browser to [http://127.0.0.1:8000](http://127.0.0.1:8000) and test the limits of your agent exactly as you'd ask a database administrator:
- *"Find the number of orders for each customer..."*
- *"Show me all electronics products out of stock"*

The application will launch the Agent natively, intercept the schema context required via its internal tools natively, and gracefully return exact PostgreSQL outputs alongside a log of the internal steps it took to retrieve it!
